## Overview

B2Bware provides a complete B2B commerce platform designed for manufacturers, distributors, and wholesalers. You get smart ordering with ERP-deep accuracy and future-proof features like tiered pricing, contract catalogs, unified product data, and real-time order and inventory sync. The platform runs directly on your ERP system, enabling seamless self-service experiences for buyers while eliminating manual processes.

## Quick Start Guide

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your account at `https://dashboard.example.com` and verify your email.
  </Step>
  <Step title="Connect ERP" icon="settings">
    Link your ERP system using our guided connector.
  </Step>
  <Step title="Configure Portal" icon="edit-3">
    Customize pricing tiers and product catalogs via the dashboard.
  </Step>
  <Step title="Launch & Test" icon="rocket">
    Publish your portal and test an order flow.

    <CodeGroup tabs="JavaScript,cURL">
      ```javascript
      const response = await fetch('https://api.example.com/v1/orders', {
        method: 'POST',
        headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
        body: JSON.stringify({
          customerId: 'cust_123',
          items: [{ productId: 'prod_456', quantity: 10 }]
        })
      });
      const order = await response.json();
      console.log(order);
      ```
      ```bash
      curl -X POST https://api.example.com/v1/orders \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "customerId": "cust_123",
          "items": [{"productId": "prod_456", "quantity": 10}]
        }'
      ```
    </CodeGroup>
  </Step>
</Steps>
