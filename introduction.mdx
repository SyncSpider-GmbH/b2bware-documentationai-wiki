## Welcome to B2Bware DataHub

B2Bware DataHub is a modular commerce API platform that organizes its functionality into specialized **Hubs**. Each hub manages a distinct domain — products, orders, customers, taxes, notifications, and more — while sharing consistent patterns for authentication, filtering, pagination, and response formats.

<Callout kind="info" title="Getting Started?">
Head to the [Quickstart](/quickstart) guide to make your first API call in under 5 minutes.
</Callout>

## Architecture

DataHub follows a hub-based architecture where each hub operates as an independent API module:

```
https://{domain}/api/v1/apps/{hub-slug}/
```

All hubs share:
- **Bearer token authentication** — a single token works across all hubs
- **Consistent filter system** — MongoDB-compatible operators via CommonFilterRepository
- **Include system** — eager loading of related resources with dot notation
- **Paginated responses** — standardized pagination on all list endpoints
