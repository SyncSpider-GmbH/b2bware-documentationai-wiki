## AI-Powered DataHub Development

B2Bware DataHub provides ready-to-use configurations for all major AI coding assistants. These skills teach your AI assistant the complete DataHub API — endpoints, filters, includes, validation rules, and best practices — so it can generate correct integration code from the first prompt.

<Callout kind="info" title="What are AI Skills?">
AI skills are configuration files that you place in your project. Your AI coding assistant reads them automatically and uses the knowledge to help you write DataHub integrations faster and more accurately.
</Callout>

## MCP Server vs. Skills Files

<Columns cols={2}>
  <Card title="MCP Server (Live Data)" icon="database">
    Connects your AI assistant to **live DataHub data**. It can browse products, search customers, create orders — all in real-time. Best for: data exploration, order management, customer lookup.
  </Card>
  <Card title="Skills Files (API Knowledge)" icon="book-open">
    Teaches your AI assistant the **API documentation**. It knows all endpoints, filters, validation rules, and can generate integration code. Best for: writing code, building integrations, debugging API calls.
  </Card>
</Columns>

<Callout kind="tip" title="Use Both Together">
For the best experience, combine the MCP server (live data access) with skills files (API knowledge). Your AI assistant can then both understand the API and interact with real data.
</Callout>
