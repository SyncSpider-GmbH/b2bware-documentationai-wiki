## Overview

The CustomerHub manages customers, companies, addresses, roles, customer groups, and notes. It integrates with AttributesHub for custom customer and company attributes and provides a settings system via HubSettingsTrait.

**Base URL:** `api/v1/apps/customer-hub/`

<Callout kind="info">
All endpoints require Bearer token authentication. See [Authentication](/authentication).
For filter syntax, pagination, and sorting, see [Common Patterns](/core-concepts/common-patterns).
</Callout>

## Examples

<Steps>
  <Step title="Search customers with includes">
    <Request tabs="cURL">
```bash
curl -s -X GET \
  'https://your-domain.com/api/v1/apps/customer-hub/customers?filter[search]=john&include=company,role,customerGroup&per_page=20&sort=-_created_at' \
  -H 'Authorization: Bearer {token}' \
  -H 'Accept: application/json'
```
    </Request>
  </Step>
  <Step title="Filter customers by company name">
    <Request tabs="cURL">
```bash
curl -s -X GET \
  'https://your-domain.com/api/v1/apps/customer-hub/customers?filter[company.name]=[{"condition":"LIKE","value":"%Acme%"}]&include=company,shippingAddresses,billingAddresses' \
  -H 'Authorization: Bearer {token}' \
  -H 'Accept: application/json'
```
    </Request>
  </Step>
  <Step title="Create a customer with attributes">
    <Request tabs="cURL">
```bash
curl -s -X POST \
  'https://your-domain.com/api/v1/apps/customer-hub/customers' \
  -H 'Authorization: Bearer {token}' \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json' \
  -d '{
    "first_name": "John",
    "last_name": "Doe",
    "email": "john.doe@example.com",
    "phone": "+1234567890",
    "salutation": "Mr.",
    "status": "active",
    "password": "securePassword123",
    "company_id": 1,
    "role_id": 2,
    "customer_group_id": 1,
    "attributes": [
      {"attribute_value_id": 10},
      {"attribute_value_id": 25}
    ]
  }'
```
    </Request>
  </Step>
  <Step title="Filter companies with VAT number">
    <Request tabs="cURL">
```bash
curl -s -X GET \
  'https://your-domain.com/api/v1/apps/customer-hub/companies?filter[vat_number]=[{"condition":"NOT EMPTY"}]&include=customers,shippingAddresses,billingAddresses&sort=name' \
  -H 'Authorization: Bearer {token}' \
  -H 'Accept: application/json'
```
    </Request>
  </Step>
  <Step title="Filter customers by date range">
    <Request tabs="cURL">
```bash
curl -s -X GET \
  'https://your-domain.com/api/v1/apps/customer-hub/customers?filter[_created_at]=[{"condition":">=","value":"2024-01-01"},{"condition":"<=","value":"2024-06-30","operator":"AND"}]&per_page=50&sort=-_created_at' \
  -H 'Authorization: Bearer {token}' \
  -H 'Accept: application/json'
```
    </Request>
  </Step>
</Steps>
