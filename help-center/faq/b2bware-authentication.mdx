title: How do I authenticate with B2Bware API?
description: Learn how to use your API key with B2Bware.
---

## Overview

All B2Bware API requests require authentication using an API Token in the header.

## Steps

1. Log in to the B2Bware Dashboard
2. Navigate to `Settings` > `API Tokens`
3. Click "Generate Token" and copy it

## Example

```bash
curl -H "Authorization: Bearer YOUR_B2BWARE_TOKEN" \
  https://app.b2bware.com/api/v1/clients
```
