## Overview

Connect SyncSpider to automate data exchange with B2Bware.

## Steps

<Steps>
  <Step title="Add integration in SyncSpider">
    ````
    Go to Integrations, search for "B2Bware", and click "Add".
    ````
  </Step>
  <Step title="Enter API Token">
    ````
    Paste your B2Bware API Token and save integration settings.
    ````
  </Step>
  <Step title="Test connection">
    ````
    Click "Test" in SyncSpider. You should see "Connection successful!"
    ````
  </Step>
</Steps>
