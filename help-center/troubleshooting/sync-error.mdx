---
title: SyncSpider Connection Error
description: Fix common connection errors with SyncSpider.
---

## Symptoms

- SyncSpider task fails
- Message: "Connection could not be established"

## Causes

- Invalid credentials
- Network issues

## Quick Fix

<Steps>
  <Step title="Verify credentials">
    ````
    Double-check your B2Bware API Token in the SyncSpider integration settings.
    ````
  </Step>
  <Step title="Test network access">
    ````bash
    curl https://app.b2bware.com/api/status
    ````
  </Step>
</Steps>
