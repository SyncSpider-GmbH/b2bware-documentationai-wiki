## Prerequisites

Before you begin, you need:
- A B2Bware DataHub instance with an active domain
- An API token (obtained through your DataHub admin panel)

## Your First API Call

<Steps>
  <Step title="Set up your token" icon="key">
    Store your API token for use in requests:

```bash
export DATAHUB_TOKEN="your-api-token-here"
export DATAHUB_DOMAIN="api.datahub.syncspider.com"
```
  </Step>
  <Step title="List products" icon="box">
    Fetch the first page of public products with pricing and images:

    <Request tabs="cURL">
```bash
curl -s -X GET \
  "https://${DATAHUB_DOMAIN}/api/v1/apps/product-hub/products?filter[status]=public&include=price,finalPrice,mainMedia&per_page=5&sort=-_created_at" \
  -H "Authorization: Bearer ${DATAHUB_TOKEN}" \
  -H "Accept: application/json"
```
    </Request>
  </Step>
  <Step title="List customers" icon="users">
    Fetch customers with their company information:

    <Request tabs="cURL">
```bash
curl -s -X GET \
  "https://${DATAHUB_DOMAIN}/api/v1/apps/customer-hub/customers?include=company,role&per_page=5&sort=-_created_at" \
  -H "Authorization: Bearer ${DATAHUB_TOKEN}" \
  -H "Accept: application/json"
```
    </Request>
  </Step>
  <Step title="List recent orders" icon="shopping-cart">
    Fetch the latest orders with line items:

    <Request tabs="cURL">
```bash
curl -s -X GET \
  "https://${DATAHUB_DOMAIN}/api/v1/apps/order-hub/orders?include=customer,orderLineItems&per_page=5&sort=-_created_at" \
  -H "Authorization: Bearer ${DATAHUB_TOKEN}" \
  -H "Accept: application/json"
```
    </Request>
  </Step>
</Steps>

## Understanding the Response

All list endpoints return paginated responses:

```json
{
  "data": [
    {
      "id": 1,
      "name": "Premium Widget",
      "sku": "WDG-001",
      "status": "public",
      "price": { "price": 29.99, "price_type": "regular" },
      "finalPrice": { "price": 24.99 },
      "mainMedia": { "media_url": "https://cdn.example.com/widget.jpg" }
    }
  ],
  "current_page": 1,
  "last_page": 10,
  "per_page": 5,
  "total": 50
}
```

## Common Operations

| Operation | Hub | Endpoint |
|-----------|-----|----------|
| List products | ProductHub | `GET /api/v1/apps/product-hub/products` |
| Create an order | OrderHub | `POST /api/v1/apps/order-hub/orders` |
| Search customers | CustomerHub | `GET /api/v1/apps/customer-hub/customers?filter[search]=term` |
| Check stock | ProductHub | `GET /api/v1/apps/product-hub/stock?filter[product_id]=...` |
| Calculate tax | TaxHub | `POST /api/v1/apps/tax-hub/calculate` |
