## Upgrade Instructions

Follow these steps to upgrade B2Bware to the latest version safely.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup of your ERP data and configurations.

````bash
# Example backup command
pg_dump -U user b2bware_db > backup_$(date +%Y%m%d).sql
````

  </Step>
  <Step title="Update Package" icon="package">

    Use your package manager to install the latest version.

    <CodeGroup tabs="npm,yarn">
````bash
npm install b2bware@latest
````
````bash
yarn add b2bware@latest
````
    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="git-branch">

    Execute database migrations.

````bash
npx b2bware migrate
````

  </Step>
  <Step title="Restart Services" icon="refresh-cw">

    Restart your B2Bware services and verify functionality.

  </Step>
</Steps>

<Callout kind="tip">
  Test upgrades in a staging environment first. Monitor logs for any `{deprecation}` warnings.
</Callout>
